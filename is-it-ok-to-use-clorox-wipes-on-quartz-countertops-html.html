Is it OK to use Clorox wipes on quartz countertops?

Answer

Most household cleaners you would usually use to do a quick clean with such as Windex, vinegar and Lysol wipes (some of which contain bleach) are not a good idea for quartz countertops. If you need to use vinegar for cleaning quartz, always make sure you dilute it with water.
